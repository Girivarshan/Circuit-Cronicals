# 🏁 Circuit Chronicles

**An Interactive Formula 1 Circuits Explorer (HTML • CSS • JavaScript)**

Circuit Chronicles is a lightweight, front-end web project that brings the 2025 Formula 1 calendar to life.
It offers a clean, interactive way to explore iconic F1 circuits, compare tracks, view rankings, and dive into driver statistics — all crafted with pure HTML, CSS, and vanilla JavaScript.

Built with simplicity in mind, this project stands as a small tribute to motorsport data, design, and the quiet beauty of well-structured code.

---

## ✨ Features

* 📅 **2025 F1 Calendar View**
  Browse all 24 circuits in chronological order with dates and rounds.

* 🔍 **Search & Filters**
  Search by circuit name, country, or round. Filter by month and sort by order or name.

* 📊 **Circuit Rankings**
  Compare tracks by speed, difficulty, and length.

* ⚖️ **Side-by-Side Comparison**
  Pick two circuits and compare their technical stats.

* 🧭 **Interactive Map (Optional Extension)**
  Visual placement of circuits across the globe.

* 🏆 **Driver Statistics**
  All-time records and notable performances.

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & responsive layout
* **Vanilla JavaScript** – Interactivity & data rendering

## 🚀 Getting Started

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/circuit-chronicles.git
   ```

2. Open `index.html` in your browser

   ```bash
   double-click index.html
   ```

That’s it. No build steps. No servers. Just open and explore.

## 🎨 Customization

* Modify colors and theme in `style.css`
* Extend features like:

  * Lap time visualizations
  * Driver comparison charts
  * Team-based statistics
